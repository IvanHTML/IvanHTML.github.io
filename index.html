Навыки
Что я использую в работе
HTML5
Именно он создает каркас сайта или приложения, а пятая версия позволит создавать более SEO-оптимизированную структуру продукта
CSS3
Этот язык стилей позволяет создавать абсолютно любой внешний вид вашего сайта или приложения. Все ограничивается только фантазией!
JavaScript
Этот язык программирования позволяет оживить все что угодно: слайдеры, окна, подсказки, вкладки, получение данных от сервера и многое другое
Jquery
Библиотека Jquery позволит ускорить разработку. Без необходимости интегрировать в проект не следует, но навык работы с ней присутствует
React
Эта библиотека позволяет создавать web-приложения
Bootstrap
Cвободный набор инструментов для создания сайтов и веб-приложений.
Gulp
Таск-менеджер для автоматического выполнения часто используемых задач
Webpack
Сборщик модулей HTML, CSS, JavaScript с открытым исходным кодом
